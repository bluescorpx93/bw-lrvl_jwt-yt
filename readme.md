<h3>JWT in Laravel</h3>

<p> To Test This </p>

- Setup Env Vars
```sh
$ cd bw-lrvl_jwt-yt
$ touch .env
```

- Update .env
```
DB_CONNECTION=<YOUR_DRIVER>
DB_HOST=<YOUR_HOST>
DB_PORT=<YOUR_PORT>
DB_DATABASE=<YOUR_DBNAME>
DB_USERNAME=<YOUR_DBUSER>
DB_PASSWORD=<YOUR_DBPASS>
```
- Run App

```sh
$ cd bw-lrvl_jwt-yt
$ composer install 
$ php artisan serve
```
